在Github上部署Ghost 0.11.9

16 June 2017

[1] https://ghost.org/developers/

以下操作在 Mac，Node 6.9.0, Ghost 0.11.9 环境完成

本地安装
  • 下载Ghost的发布版本，https://ghost.org/download/
  • 解压至本地 Ghost 文件夹，unzip Ghost-0.11.9.zip -d Ghost
  • 命令行切换至 Ghost 文件夹，通过 npm install --production 安装依赖
  • 依赖包安装完毕之后本地启动 npm start
  • 如果能看到 http://localhost:2368/ 表示启动成功
配置Ghost
  • 配置地址为 http://localhost:2368/ghost
  • 配置内容仅包含用户名、邮箱和密码
静态化

仍然使用 Buster 进行静态化

  • 安装 Buster，pip install buster
  • 配置 GitPages Repo， buster setup --gh-repo=
  • 生成静态页面，buster generate
  • 预览静态页面，buster preview，结果会提示服务端口号，默认为9000
  • 预览无误后将静态文件提交至远程目录，buster deploy
#

Done
